# CV Writing

When writing or tailoring a CV:

1. Anchor to the target role. Ask for the job description if you don't have it,
   and mirror its language and priorities.
2. Lead with impact. Each bullet = action + measurable result, not a duty.
3. Order sections by relevance to the role, not by habit.
4. Cut anything that doesn't support the target. A shorter, sharper CV wins.
5. Keep it truthful — never invent experience the user doesn't have.

Before writing a tailored CV, recall the user's full profile notes so the
bullets are grounded in their real achievements.
